Keep "inp-params.txt" file in the same folder

Compile the programs in the following manner:
    g++ SrcAssgn3-tas-CS19B1004.cpp -o tas -lpthread
    g++ SrcAssgn3-cas-CS19B1004.cpp -o cas -lpthread
    g++ SrcAssgn3-cas-bounded-CS19B1004.cpp -o cas-b -lpthread

Run the programs using:
    ./tas
    ./cas
    ./cas-b

The log files are present as TAS.txt, CAS.txt, Bounded-CAS.txt
